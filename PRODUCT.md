# Product

## Register

brand

## Users

Roblox developers and creators who already use an AI coding assistant (Claude Code, Codex,
Cursor, Windsurf, Antigravity, Cline). They are technical enough to register an MCP server and edit
a config file. They land here from a dev's recommendation, a search, or a social post, evaluating
in under a minute whether this tool is real and worth $5/month. Context: at their desk, Studio
open, impatient, allergic to marketing fluff.

## Product Purpose

Roblox MCP Pro connects an AI agent to a live Roblox Studio session so the agent can actually
build: run Luau, query and mutate the DataModel, build UI/terrain/lighting, and keep Studio in
sync with local files. The site's job is to make a skeptical developer believe it works, show what
it does concretely, and convert to a free trial / $5/month subscription. Success = trial starts.

## Brand Personality

Confident, precise, dev-native. Talks like an engineer, not a marketer: specific verbs, real tool
names, no hype. The feeling is "a sharp professional tool" (Linear/Vercel-grade craft) with a
builder's energy. Voice is plain and exact.

## Anti-references

- Generic dark-SaaS landing with a blue→purple gradient and gradient-text headline (the default AI
  output for "AI dev tool" — explicitly rejected).
- Childish / clipart "Roblox kid" aesthetic.
- Crypto / hype-bro landing (loud, vague, "to the moon").
- Boring docs-template page with no point of view.

## Design Principles

- **Show the product, don't describe it.** A concrete agent-command → Studio-result beats adjectives.
- **Earn a developer's trust with specifics.** Real tool names, exact pricing, honest refund/trial terms.
- **Restrained surface, committed accent.** Dark and quiet, with one ember accent that carries the
  whole identity. No hedging neutrals fighting the accent.
- **Every element justifies itself.** No section eyebrows, no filler cards, no copy that restates a heading.

## Accessibility & Inclusion

WCAG AA: body text ≥4.5:1, large text ≥3:1, visible focus states, full keyboard operability.
`prefers-reduced-motion` honored on every animation (crossfade/instant fallback). Color is never
the only signal.
